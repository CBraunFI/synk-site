# SYNK GROUP — Website (Entwurf)

Statische Website der SYNK GROUP für interne Abstimmung im Team.

## Live-Vorschau

Veröffentlicht über GitHub Pages: https://cbraunfi.github.io/synk-site/

## Struktur

- `index.html` — Startseite
- `about/` — Über uns / Wer wir sind
- `leistungen/` — Leadership, Change, Collaboration, Digitale Lösungen
- `qualifizierung/` — Programme & Coachings
- `wissen/` — Leada, Podcast, Wissensbereich
- `kontakt/` — Kontaktseite
- `assets/` — CSS, JS, Bilder

## Lokal ansehen

Einfach `index.html` im Browser öffnen, oder einen kleinen Server starten:

```bash
python -m http.server 8000
# dann http://localhost:8000 öffnen
```
